**What are these?**
- `fastapi` - Modern web framework for building APIs (like Express.js for Python)
- `uvicorn` - Web server to run FastAPI
- `redis` - Python library to talk to Redis cache
- `psycopg2-binary` - Python library to talk to PostgreSQL
- `sqlalchemy` - ORM (Object-Relational Mapping) - makes database queries easier
- `pydantic` - Data validation library
